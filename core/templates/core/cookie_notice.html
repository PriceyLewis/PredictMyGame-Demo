This page describes the demo cookie controls included in the portfolio build. It is not a formal compliance notice.

How Cookies Work In The Demo

The project uses cookies or local browser storage for practical app behaviour such as sessions, theme preferences, and the mock analytics preference toggle.

What The UI Demonstrates

  • Essential storage for login sessions and basic app state.
  • Preference storage for UI settings such as theme choices.
  • Optional analytics toggles that illustrate how consent controls could work in a fuller product.

Important Limitation

The cookie banner and settings modal are demo interface components. They should not be treated as evidence of audited consent handling or full regulatory compliance.
